#### 2.	방식
-	~수업 진도에 맞춰 매 스터디마다 2-3문제를 실전과 같이 시간제한을 두고 문제를 푼 뒤 접근 방법 및 풀이 공유~
- 스터디 전까지 3~5문제씩 풀어와 스터디 당일 날 코드 리뷰 (차차 난이도 상향, 시간 제한두고 풀기)
   - 수요일 스터디 : 삼성 기출 문제 2~3문제
   - 금요일 스터디 : 삼성 기출 문제 1~2문제 
- 깃헙에 스터디 외 푼 문제 링크와 코드 공유
